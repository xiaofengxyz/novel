插件名称：DEDECMS自动采集伪原创5.7正式版
插件作者：QQ:734686669
发布日期：2014年6月30日
支持版本：DEDECMS5.7
主要功能:
主要功能介绍（以下功能的实现是基于网站有流量的情况下的，任何操作都需要有一个促发器，访问网站就是一个促发器，蜘蛛来访不算）：
1、首页随机采集一篇文章，并更新首页和该文章及上个ID的文章（如果上个ID不是同栏目的话就不是上一篇文章）。
2、列表页首先更新当前列表页及下一页。（第N页就是更新N和N+1页。客户看到永远是最新的）
3、后台可以绑定栏目.请自行绑定。
4、自动生成TAG功能、自动摘要、自动关键词。
5、可设置是否自动随机推荐，特荐，幻灯，头条。
6、可以设置采集间隔时间。
7、可以设置每个规则每次采集的文章数量以及每天采集的文章总数量上限。

使用说明：
1、需要伪原创的请打开根目录下的word_replace.txt，进入采集节点管理那里导入
2、自动采集方式：
首页在底部插入 <script src="/plus/spider.php"></script>
列表页：

在/include/common.func.php
增加一下函数
function spiderpage($str)
{
$a=explode('/',$str);
if ($a)
{
$str=1;
}
return $str;
}
列表页代码如下 ：<script src="/plus/spider.php?id={dede:field.typeid/}&page={dede:pagelist listsize='0' listitem='pageno' function='trim(spiderpage(html2text(@me)))'/}"></script>
